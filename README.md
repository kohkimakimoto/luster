# Luster

A command line application framework based on Laravel.

> This product is in beta stage!

## Requirements

* PHP5.4 or later

## Installation

Create `composer.json` for installing via composer..

```
{
    "require": {
        "kohkimakimoto/luster": "5.0.*"
    }
}
```

Run composer install command.

```
composer install
```
